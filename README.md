# Portfolio 2

Welcome to my portfolio! This site showcases my JavaScript projects, lessons, and learning journey through computer science.

## About This Portfolio

This portfolio is built with Jekyll and includes:
- JavaScript lesson materials and code runners
- Homework assignments and solutions
- Projects demonstrating various programming concepts
- Personal blog posts about my learning experience

## Key Sections

- **Blog** - My JavaScript lessons and homework
- **Projects** - Showcasing various coding projects
- **About** - Learn more about me and my interests
- **Toolkit** - Resources and tools for learning

## Technologies Used

- **Frontend**: JavaScript, HTML, CSS
- **Backend**: Jekyll, Ruby
- **Tools**: VS Code, Git

## Getting Started

To run this site locally:

```bash
make dev
```

This will start a development server on localhost:4500
